# Documentación de Scripts y Configuración

Esta carpeta contiene documentación para scripts y configuraciones de Ghost.

## Archivos Disponibles

- **API_KEY.md** - Cómo obtener y configurar el Admin API Key
- **PASOS_AHORA.md** - Pasos inmediatos después de instalar Ghost
- **SEO_CODE.md** - Código HTML para SEO (Open Graph, Twitter Cards, Schema.org)
- **ADSENSE_CODE.md** - Código HTML para Google AdSense

## Uso

### Para empezar
1. Lee `PASOS_AHORA.md` para los primeros pasos
2. Configura el API Key siguiendo `API_KEY.md`

### Para configurar SEO y Ads
1. Copia el código de `SEO_CODE.md` en Ghost Admin → Settings → Code Injection → Site Header
2. Copia el código de `ADSENSE_CODE.md` en Ghost Admin → Settings → Code Injection → Site Header y Footer
