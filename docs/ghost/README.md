# Documentación de Ghost

Esta carpeta contiene toda la documentación relacionada con la configuración y uso de Ghost.

## Archivos Disponibles

- **GHOST_README.md** - Guía principal y resumen del proyecto Ghost
- **GHOST_SETUP.md** - Guía completa de instalación en servidor propio
- **GHOST_LOCAL_SETUP.md** - Guía detallada de instalación local
- **SETUP_RAPIDO.md** - Guía rápida de 5 minutos para setup local
- **QUICK_START.md** - Guía rápida para empezar a publicar
- **ghost-ads-config.md** - Configuración de Google AdSense en Ghost

## Uso Rápido

Para empezar rápidamente, lee:
1. `SETUP_RAPIDO.md` - Para instalar Ghost localmente
2. `QUICK_START.md` - Para empezar a publicar artículos

Para configuración avanzada:
- `GHOST_LOCAL_SETUP.md` - Setup local completo
- `GHOST_SETUP.md` - Setup en servidor de producción
- `ghost-ads-config.md` - Configuración de monetización
