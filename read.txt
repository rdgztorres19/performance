SEO SETUP CHECKLIST – perfstack.dev (Ghost + Cloudflare)

1) HTTPS activo y estable
   - El sitio debe abrir solo por https://perfstack.dev
   - El admin debe abrir en https://perfstack.dev/ghost
   - No usar IP ni http

2) Configurar SEO básico del sitio en Ghost
   - Ir a Ghost Admin → Settings → General
   - Definir un Title claro con keywords
   - Definir una Description atractiva (150–160 caracteres)
   - Definir idioma del sitio (en o es)

3) Añadir el dominio a Google Search Console
   - Crear propiedad de tipo Domain
   - Usar el dominio exacto: perfstack.dev
   - Elegir verificación por DNS (TXT)

4) Añadir el TXT de verificación en Cloudflare
   - Ir a Cloudflare → DNS → Add record
   - Tipo: TXT
   - Name: @
   - Content: google-site-verification=XXXXXXXX
   - TTL: Auto
   - Guardar el registro

5) Verificar dominio en Search Console
   - Esperar entre 1 y 10 minutos
   - Volver a Search Console
   - Pulsar Verify
   - Confirmar estado Verified

6) Enviar sitemap del sitio
   - En Search Console ir a Sitemaps
   - Añadir: https://perfstack.dev/sitemap.xml
   - Confirmar que Google lo acepta

7) Configurar SEO por cada artículo
   - En cada post abrir Settings (⚙️)
   - Definir SEO Title (50–60 caracteres)
   - Definir Meta Description (140–160 caracteres)
   - Revisar URL limpia sin números ni símbolos

8) Publicar contenido optimizado
   - Publicar mínimo 3 artículos iniciales
   - Cada artículo con 800–1500 palabras
   - Usar headings H1, H2, H3 correctamente
   - Enlazar artículos entre sí

9) Solicitar indexación manual
   - Ir a Search Console → URL Inspection
   - Pegar la URL del artículo
   - Pulsar Request Indexing
   - Repetir para artículos importantes
